# Template-Client
###Boilerplate React Project for A.Y. Technologies Inc. 


##Requirements

* node and npm -> `brew install node`
* webpack -> `npm install webpack -g`
* 

##Dependencies

Install all dependencies:

```
npm install
```

Adding a new development dependency:

```
npm install SOMEPACKAGE --save-dev
```

Adding a new dependency:

```
npm install SOMEPACKAGE --save
```

## Build and Bundle

To build and bundle for development:

```
webpack-dev-server

```

The hot-reload page will be loaded at 
 `http://localhost:8080`

 don't use this url... http://localhost:8080/webpack-dev-server/
